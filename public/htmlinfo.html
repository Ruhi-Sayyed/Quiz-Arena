Learn HTML

The building blocks of the web

Introduction

HTML (HyperText Markup Language) is used to structure content on the web. It defines headings, paragraphs, links, images, and more.

Why Learn HTML?

  • Essential for Web Development: Every website is built on HTML. It’s the backbone of all online content.
  • Beginner-Friendly: HTML is a great first step into coding. It’s simple and easy to grasp with no logic or syntax rules.
  • Works with CSS and JavaScript: HTML combines with CSS for design and JavaScript for interactivity. They form the core web trio.
  • Universally Supported: Every browser understands HTML. It’s a stable and long-lasting technology.

Core Concepts

  • Tags & Elements: HTML uses tags like <p> and <h1> to define elements. Tags usually come in opening and closing pairs.
  • Attributes: Elements can include extra information like classes, IDs, or links. For example, href in <a> tags defines the link target.
  • Lists & Links: Use <ul> or <ol> for lists and <a> for hyperlinks. These build structured and navigable pages.
  • Images & Media: Display media with tags like <img> and <video>. You can also embed YouTube videos with <iframe>.

Example Code

<!DOCTYPE html>
<html>
  <head><title>My Page</title></head>
  <body>
    <h1>Welcome to HTML</h1>
    <p>This is a paragraph.</p>
  </body>
</html>
