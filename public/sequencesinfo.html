Learn Sequences

Understanding patterns in numbers and letters

What Are Sequences?

Sequences are ordered lists of numbers, letters, or symbols following a specific pattern or rule. Identifying the pattern helps you find the next item(s) in the sequence.

Common Types of Sequences

  • Arithmetic Sequence: Each term is obtained by adding a fixed number (common difference).
    Example: 2, 5, 8, 11, 14 ... (common difference = 3)
  • Geometric Sequence: Each term is obtained by multiplying by a fixed number (common ratio).
    Example: 3, 6, 12, 24, 48 ... (common ratio = 2)
  • Fibonacci Sequence: Each term is the sum of the two preceding terms.
    Example: 0, 1, 1, 2, 3, 5, 8, 13 ...
  • Alphabetical Sequences: Letters arranged in a particular pattern.
    Example: A, C, E, G, I ... (every 2nd letter)

How to Solve Sequence Problems

  • Look for Differences: Calculate the difference between terms to spot arithmetic progressions.
  • Check Ratios: See if terms multiply/divide by a constant number.
  • Consider Patterns: Observe repeating or alternating patterns in numbers or letters.
  • Test Simple Rules: Try common sequence types before assuming complex rules.

Example Sequence

Find the next number in the sequence:

2, 4, 8, 16, 32, ...

Solution: This is a geometric sequence multiplying by 2 each time. Next number is 64.
