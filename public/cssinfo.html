Learn CSS

Style and design the web

Introduction

CSS (Cascading Style Sheets) controls the layout and visual presentation of HTML content. It's essential for styling websites beautifully and responsively.

Why Learn CSS?

  • Design Control: CSS allows you to change colors, fonts, spacing, and layouts. It gives full control over how your site looks.
  • Responsive Design: Make websites look good on phones, tablets, and desktops. Media queries let you adapt styles based on screen size.
  • Separation of Concerns: HTML structures content, CSS styles it. Keeping them separate makes maintenance and design easier.
  • Widely Used: Every website uses CSS. Understanding it is essential for web development and frontend jobs.

Core Concepts

  • Selectors: These target HTML elements to apply styles. Common selectors include element names, classes, and IDs.
  • Box Model: Every HTML element is a box made of margin, border, padding, and content. Understanding it is key to layout control.
  • Positioning: CSS lets you position elements using static, relative, absolute, and fixed values. This is crucial for layout customization.
  • Flexbox: A layout system for distributing space within containers. Great for aligning items horizontally or vertically.
  • Grid: A powerful two-dimensional layout system. Use it to create complex page structures easily.

Modern CSS Features

  • Variables: Reuse color codes or spacing values using --variables. Makes design more consistent and maintainable.
  • Media Queries: Apply different styles for different devices. Perfect for responsive web design.
  • Transitions & Animations: Create smooth effects when styles change. Useful for hover effects, sliders, and more.
  • Custom Fonts & Icons: Use Google Fonts and libraries like Font Awesome. Improve your design with attractive typography and visuals.

Example Code

body {
  background-color: #f9f9f9;
  font-family: Arial, sans-serif;
}

h1 {
  color: #2965f1;
  text-align: center;
}
