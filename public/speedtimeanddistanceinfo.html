Learn Speed and Distance

Mastering the Basics of Motion

Basic Concepts

Speed is the rate at which an object covers distance.

Distance is how far an object has moved.

Time is the duration taken to cover the distance.

Key Formulas

  • Speed = Distance / Time
  • Distance = Speed × Time
  • Time = Distance / Speed

Examples

Example 1:

A car travels 150 km in 3 hours. Find its speed.

Solution:

  • Speed = Distance / Time = 150 / 3 = 50 km/h

Example 2:

A runner runs at 8 km/h for 2.5 hours. How far does he run?

Solution:

  • Distance = Speed × Time = 8 × 2.5 = 20 km

Example 3:

How long will it take to travel 120 km at 40 km/h?

Solution:

  • Time = Distance / Speed = 120 / 40 = 3 hours

Example 4:

A cyclist covers 30 km in 1.5 hours. Find the speed.

Solution:

  • Speed = 30 / 1.5 = 20 km/h
